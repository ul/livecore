
# RtMidi source

This directory contains this repo's vendored copy of RtMidi. The nim module,
vendor.nim, handles the compiling and linking of these sources.

## Files

- rtmidi_c.cpp
- rtmidi_c.h
- RtMidi.cpp
- RtMidi.h
